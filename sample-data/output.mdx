# Atlas CLI

<Contents local={true} backlinks="none" depth={{"$numberInt":"2"}} class="singlecol">
  On this page
</Contents>

The [Atlas CLI](https://www.mongodb.com/docs/atlas/cli/current/) is a command line interface built
specifically for MongoDB Atlas. Use the Atlas CLI to easily
manage Atlas from the terminal.

## Manage Atlas

To perform tasks in Atlas using the Atlas CLI, see the
<Guilabel>Atlas CLI</Guilabel> tab on a task's page in this documentation. For example, to set up private endpoints using the Atlas CLI, see
the <Guilabel>Atlas CLI</Guilabel> tab on Set Up a Private Endpoint for a Dedicated Cluster.

All Atlas CLI commands in this documentation reflect the stable
version of the Atlas CLI. If you use an earlier version of
the Atlas CLI and you want to use the commands in this
documentation, you can:

Refer to the [command reference](https://www.mongodb.com/docs/atlas/cli/current/command/atlas/)
and select your Atlas CLI version from the dropdown.

[Upgrade the Atlas CLI](https://www.mongodb.com/docs/atlas/cli/current/install-atlas-cli/#update-the-atlas-cli)
to the stable version.

To view all parameters and syntax for Atlas CLI commands, see the
[command reference](https://www.mongodb.com/docs/atlas/cli/current/command/atlas/).
