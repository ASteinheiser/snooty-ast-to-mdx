      various <Ref url="query-selectors">{refs["query-selectors"].title}</Ref> to specify
      filter conditions. Use the
      [FilterDefinitionBuilder](https://mongodb.github.io/mongo-csharp-driver/2.10/apidocs/html/T_MongoDB_Driver_FilterDefinitionBuilder_1.htm)
      methods to create a filter document. For example:

      ```csharp
      var builder = Builders<BsonDocument>.Filter;
      builder.And(builder.Eq(<field1>, <value1>), builder.Lt(<field2>, <value2>));
      ```
    </Tab>
